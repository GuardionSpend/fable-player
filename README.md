# Fable Player

Тёмный музыкальный плеер для Android с интерфейсом, который подстраивается под цвет обложки играющего трека.

## Возможности

- Воспроизведение локальной музыки (MediaStore), список треков на весь экран
- Мини-плеер снизу и полноэкранный режим «Сейчас играет» с размытой обложкой на фоне
- Адаптивная тема: фон, кнопки, ползунок и акценты переливаются в цвет альбома
- Повтор (выкл / весь список / один трек) и перемешивание
- Плейлисты: создание, переименование, своя обложка, удаление (свайп вправо — боковое меню)
- Свои обложки для треков без встроенной картинки (долгое нажатие на трек)
- Поиск по названию и исполнителю
- Статистика: время прослушивания, любимый трек, размер библиотеки
- Плавные анимации: выезд экранов, кроссфейды обложек, пружинящие кнопки

Минимум Android 7.0 (API 24). Без Gradle и сторонних библиотек — чистый Java + Android SDK, APK ~58 КБ.

## Сборка

Нужны: JDK 17+, Android SDK (platform-34, build-tools 35.0.0).

```bash
./build.sh
```

Готовый файл: `FablePlayer.apk` (подписан debug-ключом, который создаётся при первой сборке).
